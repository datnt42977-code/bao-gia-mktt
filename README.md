# Báo Giá MKTT

Web app PWA soạn báo giá bê tông Mekong, xuất PDF, share Zalo.

## Cấu trúc
- `backend/` — Node.js + Express + LibreOffice (Docker, deploy Render)
- `frontend/` — Next.js PWA (deploy Vercel)

## Local dev
```bash
cd backend && npm start        # http://localhost:3000
cd frontend && npm run dev     # http://localhost:3001
```
